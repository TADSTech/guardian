# Guardian pitch outline

1. **The problem:** vulnerable Nigerians face scams, confusing healthcare information, and inaccessible digital content.
2. **The insight:** WhatsApp is already the familiar interface; support should meet people there.
3. **The product:** send text, images, links, or voice notes; Guardian explains and guides.
4. **Trust & safety:** plain language, explicit uncertainty, medical boundaries, and privacy-aware processing.
5. **How it works:** WhatsApp intake → analysis → safe explanation → support dashboard.
6. **Why it can grow:** shared contracts, independently scalable services, and an evaluation layer for local context.
